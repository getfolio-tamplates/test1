Requirements

  • Node.js 18.0.0 or higher
  • NPM or Yarn package manager
  • Basic knowledge of JavaScript/React

Installation

Local Development

# 1. Extract the downloaded files
# 2. Install dependencies
npm install

# 3. Install client dependencies
cd client
npm install

# 4. Install server dependencies
cd ../server
npm install

# 5. Return to root directory and start both servers
cd ..
npm start

Deployment

Vercel Deployment (Recommended)

# 1. Install Vercel CLI
npm install -g vercel

# 2. Login to Vercel
vercel login

# 3. Deploy
vercel --prod
Note: The project is pre-configured for Vercel deployment with the included vercel.json configuration.

Custom Server Deployment

To deploy on your own server:

  1. Set up a Node.js environment on your server
  2. Clone/upload the project files
  3. Install dependencies using npm install
  4. Set up environment variables if needed
  5. Use PM2 or similar to keep the server running: 
    npm install -g pm2
pm2 start server/index.js

Customization

Styling

The game's appearance can be customized by modifying:

  • client/src/App.css - Main styling file
  • client/src/index.css - Global styles

Game Logic

Core game logic can be found in:

  • client/src/App.js - Frontend game logic
  • server/index.js - Backend multiplayer logic

AI Difficulty

The AI logic can be modified in client/src/App.js in the makeAIMove function.
